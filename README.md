# ccs-grade-calculator
a useful grade calculator for the students of ccs!

📊 CCS Grade Calculator

Made by Joshua Smith, coded by ChatGPT

A powerful, browser-based grade calculator designed for weighted category classes.
Built to help students track grades, predict outcomes, and reduce stress.

🚀 Features
📥 Gradebook Import
Paste your gradebook directly from your school system
Automatically creates:
categories
weights
grades
Missing assignments are counted as 0 / max
🧮 Weighted Grade Calculation
Supports category weights (Assignments, Quizzes, Tests, etc.)
Uses points earned / points possible
Ignores empty categories until they have grades
🔮 Future Grade Solver
Add upcoming assignments, quizzes, or tests
Set a target grade
See:
best case (100s)
worst case (0s)
exact scores needed
🎯 Equal Score Calculator
Shows what grade you need on all upcoming assignments if they were equal
📉 Minimum Score Per Assignment
Shows the lowest score you can get on each upcoming assignment
(assuming all others are perfect)
🧠 Sample Combinations
Generates example grade combinations that reach your target
⚠️ Lowest Safe Score Tool
Find the lowest score you can get on a specific assignment
Ensures your overall grade doesn’t drop below a target
🔁 Drop Lowest Grade Support
For upcoming grades, you can:
check “Drop lowest in category”
Simulates teachers replacing your lowest grade automatically
📝 Notes System

Add personal notes like:

“assuming I fix that quiz”

Saved with each class
Includes:
save button
delete all (with confirmation)
💾 Save / Load Classes
Save multiple classes locally in your browser
Load them anytime
Delete when no longer needed
🌙 Dark Mode
Toggle between light and dark mode
Automatically saves your preference
💻 How to Use
(Optional) Paste your gradebook and click Import
Add or edit categories if needed
Add grades manually if needed
Set a target grade
Add upcoming assignments
Click Solve

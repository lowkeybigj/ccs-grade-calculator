# ccs-grade-calculator
a useful grade calculator for the students of ccs!

CCS Grade Calculator

Made by Joshua Smith, coded by ChatGPT

A browser-based grade calculator for weighted classes. Helps you track
grades, predict outcomes, and plan ahead.

FEATURES

Gradebook Import - Paste your gradebook directly from your school
system - Automatically creates categories, weights, and grades - Missing
assignments are counted as 0 / max

Weighted Grade Calculation - Supports category weights (Assignments,
Quizzes, Tests, etc.) - Uses points earned / points possible - Ignores
empty categories until they have grades

Future Grade Solver - Add upcoming assignments, quizzes, or tests - Set
a target grade - Shows: - best case (100s) - worst case (0s) - exact
scores needed

Equal Score Calculator - Shows what grade you need on all upcoming
assignments if they were equal

Minimum Score Per Assignment - Shows the lowest score you can get on
each upcoming assignment - Assumes all other upcoming grades are perfect

Sample Combinations - Generates example grade combinations that reach
your target

Lowest Safe Score Tool - Find the lowest score you can get on a specific
assignment - Keeps your overall grade above a target

Drop Lowest Grade Support - Option to drop the lowest grade in a
category for upcoming scores

Notes System - Add personal notes (example: assuming I fix that quiz) -
Includes save and delete buttons - Saves with each class

Save / Load Classes - Save multiple classes locally in your browser -
Load them anytime - Delete when no longer needed

Dark Mode - Toggle between light and dark mode - Saves your preference

HOW TO USE

1.  Paste your gradebook (optional)
2.  Add or edit categories
3.  Add grades if needed
4.  Enter a target grade
5.  Add upcoming assignments
6.  Click Solve

NOTES

-   Data is stored locally in your browser
-   Clearing browser data deletes saved classes
-   Uses standard A/B/C/D/F grading

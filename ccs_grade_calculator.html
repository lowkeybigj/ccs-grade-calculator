Class Setup

Type a name for your class if you want to save it. Click Save Class to keep it. Load Selected Class brings it back later. Delete Selected Class removes it.
Saved locally in your browser.

Paste Gradebook Import

Paste your gradebook text here exactly how it copies from your school page. Missing grades will count as zeroes. Importing will replace your current categories and grades.

Add Category

Make one category for each type of grade, like Assignments, Quizzes, or Tests. Put how much each one is worth. All of them together should equal 100.

Categories

This shows all your categories. Add your grades here using points earned and total points. You can also edit or delete anything here.

Grades Notes

Use this to write notes to yourself, like assumptions you made or grades you changed. These notes save with the class.

Current Grade Summary

This shows your current grade, your letter grade, and how each category is affecting your grade.

Target / Future Grade Solver

Use this to see what grades you need in the future. Add your upcoming grades, then click Solve.

Upcoming Grades

Add a slot for each future grade. Choose what category it is in and how many points it is worth. You can name it if you want. You can also check Drop lowest in category if this new grade would replace the lowest grade in that category.

Lowest Safe Score

Use this to find the lowest score you can get on one future grade without your class grade dropping below a target. Pick the category, enter the points possible, type the class grade you want to stay above, then click Find Lowest Safe Score.

Solver Results

This shows your results. It includes best case, worst case, what you need on each grade, and some example score combos.

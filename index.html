Overview

This calculator is built for category-weighted classes. That means different types of work, like assignments, quizzes, and tests, do not all affect your grade equally. Instead, each category has a weight, and the calculator uses those weights to figure out your real class average.

If your school gradebook works with categories and percentages, this calculator is meant to copy that logic as closely as possible. It can help you understand your current grade, what is dragging it down, what future scores you need, and how much your grade could improve if you fixed missing work or corrections.

Class Setup

Use Class Setup to name the class and optionally set a goal grade. Saving the class is important because the Analyzer and Planner tools only work with saved classes.

What to do here: type the class name, type a goal grade if you want one, then click Save Class. If you later want to load an old class, choose it from the saved classes dropdown and click Load Selected Class.

Share this class creates a link for the currently loaded class. That is useful if you want to send the setup to someone else or keep a quick copy of that one class.

Importing a Gradebook

This is the fastest way to build a class. Paste your copied gradebook text into the import box, then click the import button. The calculator will try to detect category names, weights, assignments, missing grades, and points.

Missing grades are counted as zeroes, because that is how most gradebooks treat missing work until you turn it in. If your import looks wrong, check the Grades tab afterward and clean up anything unusual there.

This is especially useful when you have a lot of assignments and do not want to enter everything by hand.

Categories

Categories are the buckets that your grades go into. Common examples are Assignments, Quizzes, and Tests. Each category has a weight that tells the calculator how important that category is to the final average.

If you add categories manually, make sure the weights are realistic for your class. The calculator can still work if the weights do not add to exactly 100, but your summary will warn you if the setup looks off.

When you leave the correction cap blank, the calculator can now choose smart defaults: assignments get no cap, quizzes default to 80, and tests default to 70.

Grades

The Grades tab is where you actually manage the class. You can add assignments, edit assignments, delete them, mark them as missing, and apply correction settings. This is also where imported grades appear after you use the import tool.

Each assignment stores points earned and points possible. If you edit a missing assignment, you can now also set a late penalty to show what the assignment would realistically count for if you eventually turn it in late.

If you are confused why your grade looks high or low, this is usually the first page you should check.

Corrections

Corrections let you model classes where low grades can be fixed up to a certain cap. For example, a corrected quiz might only be allowed to rise to 80. Correction caps are set per category.

When a grade is correction-eligible, you can turn correction on for that grade. The calculator will then use the corrected score instead of the original one. This matters in the Summary, Insights, and Planner features.

Corrections are treated as opportunities to improve existing grades. Missing work is treated separately.

Insights

The Insights tab tries to explain your class instead of just showing numbers. It can point out what is hurting you the most, what your weakest category is, whether your current setup is risky, and what kinds of moves help most.

This is the page to use when you want guidance, not just calculation. It is especially useful when your class has many assignments and you want to know what deserves attention first.

Solver

The Solver is for future work. Use it when you want to know what scores you need on assignments that are not graded yet.

Upcoming Grades lets you add future assignments one by one. You choose the category, points possible, and optional label. If your teacher drops the lowest grade in a category, you can use the drop-lowest option there too.

Solve estimates what scores you need across those future assignments to hit your target. This is helpful when you want to answer questions like, “What do I need on the next two quizzes to keep an A?”

Lowest Safe Score answers a different question: “What is the lowest score I can get on this one assignment and still stay above a chosen grade?” This is more of a safety check than a target planner.

Planner

The Planner looks across saved classes instead of just the one you are currently editing. It ranks the best moves you can make, like turning in a missing assignment or correcting a low grade.

This is useful when you have several classes and you want to know where your time will matter most. Instead of guessing which assignment is most worth doing, the Planner compares the estimated gains for you.

If a class does not show useful moves, that usually means it either has no missing work, no correctable grades, or no saved correction caps.

Max Possible Grade

The Max Possible Grade tool is in the Planner section. It lets you choose a saved class and see the highest grade you could reach if you turned in every missing assignment, got full credit on all saved upcoming work, and used every available correction.

This is a “best realistic case” tool, not a guarantee. It now also uses late penalties on missing assignments, so the result is more realistic than simply assuming every missing grade turns into a perfect score.

Themes

The theme dropdown changes the overall color style of the app, and the dark mode toggle changes whether you are using the light or dark version of the selected theme. Your selected theme should persist after reloads.

Themes do not affect the calculations. They are only there to make the app easier or more fun to use.

Report

The Report tab creates a copy-ready grade report. Choose which saved classes to include, turn AI mode on if you want help from an assistant, add extra context, then generate and copy the report.

Paste your gradebook exactly how it copies from your school page. Missing grades count as zeroes. Importing replaces your current categories and grades.

Add Category

Make one category for each type of grade. Set the weight, and if needed, a correction cap percent for that category. Grades below 70 are not correctable.

Categories

This shows all your categories and assignments. You can add assignment names, correct eligible grades, edit or delete grades, and change category rules.

Current Grade Summary

This shows your current grade, your letter grade, and how each category is affecting your grade.

Insights & Warnings

This section shows what is hurting your grade the most, whether common grade targets are still reachable, and warnings about risky scores.

Upcoming Grades

Add a slot for each future grade. Choose its category, points possible, and optional name. You can also check Drop lowest in category if the new grade would replace the lowest grade there. If a saved upcoming assignment has effectively become missing, use the Missing option so the max-grade tool does not assume you will get 100% on it.

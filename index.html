Class Setup

Type a class name if you want to save it. Goal grade is optional. Save each class so the all-classes planner can use it.
Saved locally in your browser.

All Classes Planner

This scans your saved classes and shows the best missing assignments and corrections to raise grades. It uses assignment names and saved goal grades when available.
Nothing showing up? Try setting the correction cap in each of your categories in the grades tab.

Paste Gradebook Import

Paste your gradebook exactly how it copies from your school page. Missing grades count as zeroes. Importing replaces your current categories and grades.

Add Category

Make one category for each type of grade. Set the weight, and if needed, a correction cap percent for that category. Grades below 70 are not correctable.

Categories

This shows all your categories and assignments. You can add assignment names, correct eligible grades, edit or delete grades, and change category rules.

Current Grade Summary

This shows your current grade, your letter grade, and how each category is affecting your grade.

Insights & Warnings

This section shows what is hurting your grade the most, whether common grade targets are still reachable, warnings about risky scores, and a small trend graph based on your assignment order.

Target / Future Grade Solver

Use this to see what grades you need in the future. Add upcoming grades, then click Solve.

Upcoming Grades

Add a slot for each future grade. Choose its category, points possible, and optional name. You can also check Drop lowest in category if the new grade would replace the lowest grade there.

Lowest Safe Score

Use this to find the lowest score you can get on one future grade without your class grade dropping below a target.

Solver Results

This shows your results. It includes best case, worst case, what you need on each grade, and example score combinations.
